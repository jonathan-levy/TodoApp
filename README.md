# TodoApp

You must have npm and node js installed in order to launch the server.
<br />
$ git clone https://github.com/jonathan-levy/TodoApp.git <br />
$ cd TodoApp <br />
$ npm install <br />
$ node app

<br />
<br />

Then go with your browser to this url: localhost:3000/todo
